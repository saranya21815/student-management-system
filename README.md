# Student Management System (Python)

## Description
A simple console-based Student Management System built using Python. It performs CRUD operations and stores data permanently using file handling.

## Features
- Add student records
- View all students
- Search student by ID
- Delete student record
- Find topper student
- Generate report (average, pass/fail)

## Technologies Used
- Python
- File Handling (txt storage)
- Lists and Dictionaries
- Functions
